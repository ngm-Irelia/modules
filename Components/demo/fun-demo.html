Components 现有 工具方法 介绍：
Components.selectText()
禁止文字选择
Components.loadScriptPromise(url)
动态加载js
@param url 需要加载的js文件路径
Components.getTime(time)
对时间的相关处理
@param time 字符串 { '' | 'now' | 'day' | 'week' | 'month' | 'year' | 数字 }
'' ：当前日期，包含时分秒
'now' ：当天
'day' ：一天前
'week' ：一周前
'month'：一月前
'year' ：一年前
number ：number天前。 为负数的话，就是 n天后
Components.getData(urls, param, type, tradit)
使用promise对ajax接口的封装
@param urls 请求的url
@param param 请求的参数
@param type 请求的类型GET POST 等
@param tradit 是否自动解析数组
Components.getSearch(search)
解析search的值，转为json对象
@param search window.location.search
Components.getCookie(search)
解析cookie的值，转为json对象
Components.CheckEmail()
邮箱监测
@param email 邮箱账号
Components.checkIDCard()
身份证检测
@param card 身份证号
Components.trim()
去除两侧空格
@param v 要处理的字符串
Components.isIE()
检测是否是IE、Edge浏览器
Components 现有 实例方法 介绍：
Components().BarChart.run(showId, data)
组件-横向柱状图
@param showId 放组件的盒子id
@param data 现实的数据. 例子：
[{ id:"111", name:"中国", number:1399 },{ id:"999", name:"中国2", number:999 }]
Components().PictureVirtual.run(showId, Img)
图片虚化组件 -注意，现在要求 父盒子必须要有宽高~，后续有时间，在代码中优化下组件
@param showId 放组件的盒子id
@param Img 图片
Components().PictureVirtual.run()
图片虚化组件-通过class【picture-virtual】定义
父元素上设置 class="picture-virtual" data-src="imgurl"
运行： Components().PictureVirtual.run(); 即可

若是动态加载的DOM元素，需要在元素加载完成后再次调用 run()
Components().loading(id,clean)
loading加载组件
@param id 选填，需要加载的父元素 如果为空，则全页面覆盖的loading，不为空则为局部loading
@param {boolean} clean 选填； false清除loading，true和 空 ，添加loading
运行： Components().loading(参数)

Components().alert(config)
提示信息：alert弹框组件
@param config 一个对象 or false ； 为false，去掉弹框
支持的字段有
{
title:"弹框标题-支持html"
content:"弹框内容-支持html"
color:标题文字颜色
time:弹框过time时间后，隐藏
}
使用方式：
Components().alert({ title:"标题", content:"弹框内容-支持html",color:'red' });

Components().notice(config)
公告栏：notice组件
@param config
支持的字段有
{
title:"公告标题-支持html",
content:"公告内容-支持html",
color:"公告文字颜色{string}",
speed:"字条滚动速度，毫秒{number}",
size:"循环滚动次数{number}",
dom:'domid{string}'
}
使用方式：
Components().notice({ title:"标题:", content:"弹框内容-支持html", color:"red", speed:10, size:2, dom:'notice' });

Components().modal(config)
弹框：modal组件
@param config
支持的字段有
{
title, // 弹框标题-支持html
content, // 弹框内容-支持html
btn=[], // 按钮
btnEvent=[], // 按钮事件
area=['300px','190px'], // modal区域
id="modal-"+new Date().getTime() // modal 唯一 id
}
@return 返回modal实例

使用方式：

let saveModal = Components().modal({
title:"提示",
content:"模块顺序发生变化，是否保存？",
btn:["是","否"],
btnEvent:[
function(){//是 的回调函数
  saveModal.remove();
  //在这做些什么~
},
function(){ //否 的回调函数
  saveModal.remove();
  //在这做些什么~
}
]
});
提供方法： saveModal.remove() 删除modal弹框
